# Information Gathering Methodology

To ensure the AI Resume Analyzer solves real-world pain points, information was gathered using the following framework:

1. **Secondary Market Research:** Evaluated existing open-source ATS parsers and commercial tools (e.g., ResumeWorded, Jobscan) to identify feature gaps.
2. **Technical Benchmarking:** Tested the DeepSeek API against raw PDF extraction texts to validate latency, token cost, and the reliability of JSON-structured outputs.
3. **Qualitative Interviews:** Conducted structured discussions with peers navigating the current tech job market.
4. **Quantitative Surveys:** Distributed a targeted questionnaire to CS students to map standard failure points in application pipelines.
