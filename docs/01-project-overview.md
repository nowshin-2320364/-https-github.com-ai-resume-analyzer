# Project Overview: AI Resume Analyzer

## Executive Summary
The AI Resume Analyzer is an intelligent web application designed to help job seekers optimize their resumes against specific job descriptions. By leveraging the DeepSeek API, the system transcends traditional keyword-matching Applicant Tracking Systems (ATS) to provide deep semantic gap-analysis, actionable feedback, and dynamic bullet-point rewriting.

## Core Value Proposition
Traditional ATS checkers often penalize candidates for formatting anomalies or exact-match keyword failures. Our solution reads unstructured data semantically, mimicking a human recruiter's comprehension while maintaining the speed and scale of an automated system. 

## Project Scope
The project is architected as a decoupled full-stack application:
- **Frontend:** React.js (Vite) for responsive client interaction.
- **Backend:** FastAPI (Python) for orchestration and PDF parsing.
- **AI Engine:** DeepSeek API for LLM-powered structural evaluation and JSON data extraction.
