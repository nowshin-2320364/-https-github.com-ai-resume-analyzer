# Acceptance Criteria

- **User Story 1 (PDF Upload):** The system must accept `.pdf` files up to 5MB. It must reject `.docx`, `.jpg`, or other formats with a clear error message.
- **User Story 2 (JD Input):** The text area must accept up to 10,000 characters of raw text.
- **User Story 3 (Match Score):** The AI must return an integer value strictly between 0 and 100.
- **User Story 4 (Rewrites):** The system must identify at least 1 weak bullet point and return the original text, improved text, and a short reasoning string.
