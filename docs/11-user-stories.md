# User Stories

1. **As a job seeker**, I want to upload my resume as a PDF, so that I don't have to manually copy and paste my document text.
2. **As a job seeker**, I want to paste a job description, so that the AI can evaluate my fit for a specific role.
3. **As a job seeker**, I want to see a numerical match score, so that I can quickly gauge my baseline compatibility.
4. **As a job seeker**, I want the system to suggest better phrasing for my experience section, so that my achievements sound more impactful to recruiters.
